# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/rahulekhar18-code/SmartDine-AI.git
```

```bash
cd SmartDine-AI
```

-----------------------------------------------------

## Backend Setup

```bash
cd backend
npm install
```

Create `.env`

```env
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
PORT=5000
```

Run migrations:

```bash
npx prisma migrate dev
```

Start backend:

```bash
npm run dev
```

--------------------------------------------------------------

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
